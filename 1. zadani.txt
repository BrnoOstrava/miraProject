Projekt Kalkulacka
******************

- Toto bude perfektní ukázka OOP. Pokud budem všechno dělat dobře, tak uvidíš krásný update v praxi;)
- Definuj třídu Kalkulacka a v ní tyto metody:
	- int scitani(int, int)
	- int odcitani(int, int)
	- int nasobeni(int, int)
	- double deleni(int, int)
	- String deleniSeZbytkem(int, int)
	- String prevodDoDvojkoveSoustavy(int)
	- int prevodDoDesitkoveSoustavy(String)
	- int mocnina(int, double)
	- int mocninaNaDruhou(int)
	- double odmocnina(int)
	- int faktorial(int)
- A nakonec v tomto projektu definuj třídu GrafickaKalkulacka, ve které zatím nic nemusíš dělat, jen ji vytvoř a nech prázdnou
